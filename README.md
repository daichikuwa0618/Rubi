# Rubi

[![Language](https://img.shields.io/badge/language-Swift%205.0-orange.svg)](https://swift.org)

## Description
「Rubi」は入力された文字列を日本語に変換する翻訳アプリです。翻訳機能のAPIは「ひらがな化API」を使用しています。
